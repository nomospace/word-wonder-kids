# Word Wonder Kids H5

> 儿童英语单词学习应用 - 基于 Angular 17

[![Build Status](https://github.com/nomospace/word-wonder-kids/actions/workflows/build.yml/badge.svg)](https://github.com/nomospace/word-wonder-kids/actions)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 简介

专为儿童设计的英语单词学习应用，支持 Web 和移动端。通过游戏化方式帮助孩子学习核心英语单词。

## 功能特性

- 🎓 年级选择（学前班 - 六年级）
- 📖 单词卡片（图文 + 发音 + 例句）
- 🎮 四种游戏（连连看、听音选词、拼写挑战、单词测验）
- 📈 学习进度追踪
- 🔊 TTS 发音
- 📱 响应式设计

## 快速开始

```bash
cd app
npm install
npm start
```

访问 http://localhost:4200

## 构建

```bash
npm run build
```

## 技术栈

- Angular 17+
- TypeScript
- RxJS
- SCSS
- Web Speech API

## 词库来源

- Oxford 3000
- Dolch Sight Words
- Fry Words
